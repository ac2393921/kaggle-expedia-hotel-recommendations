# Expedia Hotel Recommendations

## About

kaggleのExpedia Hotel Recommendationsを実験用リポジトリ

URL: https://www.kaggle.com/competitions/expedia-hotel-recommendations

## Useage

- 学習を実行

```bash
make up
docker compose exec mlflow bash
python3 src/main.py
```

- Mlflowを起動
http://localhost:5000/#/
